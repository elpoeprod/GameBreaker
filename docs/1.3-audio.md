# GBAudio

## Definition
```c++
#ifndef GB_DONT_USE_SFX
typedef struct GBAudio {
    _GB_AudioType_ chunk;
    double vol;
    double pos;
    int type;
    double x,y,z;
    double pan;
    int handle;
    double len;
} GBAudio;
#endif
```

`GBAudio` is a type which allows to store audio data in a program.
